Sobre o Projeto

Este site apresenta um gerador de imagens baseado em inteligência artificial.

O objetivo do projeto é permitir que usuários criem imagens personalizadas a partir de descrições de texto, de forma simples e intuitiva.

Tecnologias utilizadas:

  1. HTML para a estrutura do site
  2. CSS para o layout e estilo visual
